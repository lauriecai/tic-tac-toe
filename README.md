## Tic Tac Toe
A simple tic-tac-toe game for 2 players.

### Code Structure
1. Factory function `createPlayer`
- Generate `playerOne` and `playerTwo` objects

2. `gameBoard` object
- Generate board array
- Add event listeners on each square

3. `game` object
- `checkWinner()`
- `alertNextPlayer()`
- `nextPlayer()`
- `declareTie()`
